napari-bigwarp

by Ben Kantor
Active New

Plugin Description

BigWarp-like interface for napari. This plugin provides interactive registration and alignment capabilities similar to the popular BigWarp tool in Fiji/ImageJ. Features include:

  • Interactive landmark placement
  • Real-time transformation preview
  • Multiple transformation types (affine, thin-plate spline)
  • Export functionality for transformation matrices
  • Support for 2D and 3D image registration

Installation & Quick Start

Install via pip:

pip install napari-bigwarp

Or install in napari via the plugin manager.

Quick Start: Open napari, load two images, then access BigWarp from the Plugins menu.
